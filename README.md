Title: This Is Where It Begins (TIWIB for short)
About: This is Where It Begins is an interactive game, where you the player have to control an unknow person in an lab. Your goal is to collect all journals entries, to unlock the truth of the story. If you collect all of them, then you get the good ending. If you, missed one, or don't collect all of them, you get the bad ending, not revealing the truth of the story.
resources: Use GameMaker Studios 2 for assets, use react/vite for the frontend, use node for backend and use firebase for database 
Controls: Use mouse to interact with object/buttons (or you can press E to interact with objects), use WASD to control your players
